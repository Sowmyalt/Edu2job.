#  Edu2Job

Edu2Job is a data-driven career guidance platform that helps students transition from education to employment through personalized career predictions and structured learning roadmaps.

##  Features
- Career prediction based on interests and skills
- Skill gap analysis for target job roles
- Personalized learning roadmaps
- Insights into emerging job roles and industry trends

##  Tech Stack
- Frontend: React
- Backend: Django 
- Database: PostgreSQL 
- AI/ML: Python (Random Forest algorithm)
